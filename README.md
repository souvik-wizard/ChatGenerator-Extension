# Installation

NODE VERSION SHOULD BE >= 18
<br>

```bash
npm i
```

# Start the server

```bash
npm run dev
```

# Add your API key (Gemini API by Google)

```bash
/entrypoints/content.ts

const apiKey = "ENTER YOUR API KEY HERE";
```
